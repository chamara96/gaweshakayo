TREE PLANTATION PROGRAM 2019

  • By: Gaweshakayo
  • 23 November 2019

අප රටතුල ප්‍රථම වතාවට විශ්වවිද්‍යාලයක් තුල වන සියලුම විද්‍යාර්ථයින් විසින් පීඨ පරිශ්‍රය තුළ රුක් රෝපණය කිරීමේ නියමු ව්‍යාපෘතිය රුහුණ විශ්වවිද්‍යාලීය ඉංජිනේරු පීඨ පරිපාලනය, රුහුණ විශ්වවිද්‍යාලීය ඉංජිනේරු පීඨ ශිෂ්‍ය සංගමය හා "ගවේෂකයෝ" සංසදය එක්ව 2019 නොවැම්බර් 23වන දින සිදුකිරීමට සියළු කටයුතු සූදානම් කර අැත. එහි අාරම්භක පියවරක් ලෙස පීඨ පරිශ්‍රයේ අාක්‍රමණික ශාක ඉවත් කර එ් වෙනුවට දේශීය පැල 1000කට අධික ප්‍රමාණයක් මුළු පීඨ පරිශ්‍රය පුරා රෝපණය කිරීමට සැලසුම් කර අැත. තමා විසින් රෝපණය කරනු ලබන පැලය තම සරසවි ජීවිතය තුල හොදින් රැකබලා ගැනීම හා සරසවියෙන් ඉවත්වන දිනයක නවක සහෝදරත්වයකට තම පැලයේ වගකීම පැවරීම මගින් ශාක සංරක්ෂණ, අාරක්ෂා කිරීම මෙහිදී සිදුවන සුවිෂේශී කාර්ය්‍යකි. තමා විසින් රැකබලාගන්නා පැලය අධීක්ෂණ කිරීමේ පහසුව සදහා වෙබ් යෙදුමක්( web application ) එළිදැක්වීම එදිනදී සිදුවේ.

රුහුණ විශ්වවිද්‍යාලයීය ඉංජිනේරු පීඨ පරිපාලනය,රුහුණ විශ්වවිද්‍යාලයීය ඉංජිනේරු පීඨ ශිෂ්‍ය සoගමය හා පීඨ 'ගවේශයකෝ' සoගමය එක්ව සoවිධානය කරන ලද රුක් රෝපණ ව්‍යාපෘතියේ සමාරම්භක උළෙල 2019 නොවැම්බර් 23 දින පීඨයේ නුග පියසේ දි පීඨාධිපතිතුමන් ප්‍රමුඛ ආචාර්ය මණ්ඩලයෙන්ද ව්‍යාපෘතිය සඳහා අනුග්‍රහය සැපයු Creative software ආයතනයේ මහත්ම මහත්මීන්ගෙන් හා සියලුම වසරවල සහෝදර සහෝදරියන්ගේ සහභාගිත්වයෙන් පැවැත්වුනි.පීඨාධිපතිතමන්ගේ හා creative software ආයතනයේ විධායක කළමනාකාරතුමන් පැළ රෝපණය කිරීමෙන් අනතුරුව සියලු සහෝදරත්ව එක්ව පීඨ පරිශ්‍රයේ තමාට නියමිත ස්ථානගේ පැළ රෝපණය කරන ලදී. ඒ අනුව එක දිනක් තුල සියලු සහෝදරත්ව එකව පැළ 1000 කට අධික ප්‍රමාණයක් මව් දෙරණට එක්කරන ලදී.
